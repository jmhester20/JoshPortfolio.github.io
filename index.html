Fitness Log App

This fitness log app is designed to help users stay consistent and accountable with their workouts. Users can log daily sessions, track progress over time, and easily view past workouts by selecting specific dates. The frontend is built with React for a responsive and interactive user experience, while the backend runs on Flask to handle data and authentication. MongoDB is used for flexible and scalable data storage, and JWT is implemented to ensure secure user login and session management. With the apps interface and performance, it's simple to stay on top of your fitness journey.

Virginia Tech Quiz

An interactive single-page quiz built with HTML5, CSS and JavaScript. Users choose from four VT-themed categories (History, Football, Basketball, Traditions) and pick the number of questions, then tackle timed questions delivered in random order. Real-time feedback is provided via CSS transitions and dynamic class toggling, while a JavaScript-powered countdown timer and score tracker keep the experience engaging.

FFMI Calculator

The FFMI calculator is a responsive web app built with HTML, CSS, and JavaScript that estimates lean muscle mass based on height, weight, and body fat percentage. It supports both metric and imperial units for the calculation. I built it because I'm passionate about bodybuilding and wanted to combine that interest with web development.
